useSplashScreen.tsx

왼쪽은 실제 코드, 오른쪽은 같은 state가 어디에서 만들어지고 변경되고 반환되는지 정리한 영역입니다.

Code

1import { useEffect, useState } from 'react';
2
3export type SplashScreenPhase = 'loading' | 'intro' | 'fadeOut';
4
5const FORCE_LOADING_TEST = true;
6
7const LOADING_TEST_TIME = 1000;
8const INTRO_TIME = 1000;
9const FADE_OUT_TIME = 800;
10
11function useSplashScreen() {
12 const [showSplashScreen, setShowSplashScreen] = useState(true);
13
14 const [splashScreenPhase, setSplashScreenPhase] =
15 useState<SplashScreenPhase>(() => {
16 if (FORCE_LOADING_TEST) return 'loading';
17
18 return document.readyState === 'complete' ? 'intro' : 'loading';
19 });
20
21 useEffect(() => {
22 if (splashScreenPhase !== 'loading') return;
23
24 if (FORCE_LOADING_TEST) {
25 const loadingTimerId = window.setTimeout(() => {
26 setSplashScreenPhase('intro');
27 }, LOADING_TEST_TIME);
28
29 return () => {
30 window.clearTimeout(loadingTimerId);
31 };
32 }
33
34 const handleLoad = () => {
35 setSplashScreenPhase('intro');
36 };
37
38 window.addEventListener('load', handleLoad);
39
40 return () => {
41 window.removeEventListener('load', handleLoad);
42 };
43 }, [splashScreenPhase]);
44
45 useEffect(() => {
46 if (splashScreenPhase !== 'intro') return;
47
48 const introTimerId = window.setTimeout(() => {
49 setSplashScreenPhase('fadeOut');
50 }, INTRO_TIME);
51
52 return () => {
53 window.clearTimeout(introTimerId);
54 };
55 }, [splashScreenPhase]);
56
57 useEffect(() => {
58 if (splashScreenPhase !== 'fadeOut') return;
59
60 const removeTimerId = window.setTimeout(() => {
61 setShowSplashScreen(false);
62 }, FADE_OUT_TIME);
63
64 return () => {
65 window.clearTimeout(removeTimerId);
66 };
67 }, [splashScreenPhase]);
68
69 return {
70 showSplashScreen,
71 splashScreenPhase,
72 };
73}
74
75export default useSplashScreen;

showSplashScreen 연결

showSplashScreen은 화면에 SplashScreen을 보여줄지 결정하는 state입니다.

12
생성: 처음에는 true입니다. 즉 SplashScreen을 보여주는 상태로 시작합니다.
61
변경: phase가 'fadeOut'이 된 뒤 일정 시간이 지나면 false로 바뀝니다.
70
반환: App 컴포넌트가 이 값을 받아서 SplashScreen을 렌더링할지 판단합니다.
const { showSplashScreen, splashScreenPhase } = useSplashScreen();

splashScreenPhase 연결

splashScreenPhase는 SplashScreen이 지금 어떤 단계인지 나타냅니다.

  1. 'loading': 로딩 화면
  2. 'intro': 로딩 이후 소개 화면
  3. 'fadeOut': 사라지는 단계

흐름은 loading → intro → fadeOut → 제거입니다.

return의 의미

아래 코드는 state를 새로 만드는 코드가 아닙니다. hook 내부의 값을 바깥 컴포넌트가 사용할 수 있게 내보내는 코드입니다.

return { showSplashScreen, splashScreenPhase, };

거름망 역할

if (splashScreenPhase !== ... ) return;은 지금 단계가 아니면 해당 effect를 바로 종료시키는 조건문입니다.

  • loading effect는 loading 단계에서만 실행
  • intro effect는 intro 단계에서만 실행
  • fadeOut effect는 fadeOut 단계에서만 실행

색상 기준

  • showSplashScreen: 보여줄지 여부 값
  • setShowSplashScreen: 보여줄지 여부를 바꾸는 함수
  • splashScreenPhase: 현재 단계 값
  • setSplashScreenPhase: 현재 단계를 바꾸는 함수
